```shell
Usage of govader:
	govader [flags] -type T [directory]
	govader [flags] -type T files... # Must be a single package
For more information, see:
	https://github.com/AhmadWaleed/go-validation
Flags:
  -locale string
    	locale to use for error messages; default en (default "en")
  -output string
    	output file name; default srcdir/<type>_schema.go
  -type string
    	comma-separated list of type names; must be set
```
